# LaClave

Verificador personal de noticias. Proyecto personal, sin ánimo comercial.

La entrada es una noticia (enlace, texto completo o resumen); la salida, un informe de veracidad con nota 1-10, veredicto afirmación por afirmación, fuentes de contraste y, cuando hay evidencia, el origen probable del bulo.
